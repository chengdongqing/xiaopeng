# xiaopeng

#### 精仿小鹏官网，基于 vue+vite+typescript

#### 预览：[小鹏官网](https://chengdongqing.gitee.io/xiaopeng)

#### 源码： [xiaopeng](https://gitee.com/chengdongqing/xiaopeng)

#### 项目配置

- 核心框架： Vue 3
- 构建工具： Vite 4
- node 版本：Node.js 18
- 开发工具： WebStorm 2023.2

#### 使用方式

    安装：yarn install (可简化为：yarn)
    运行：vite dev (可简化为：vite)
